Portal 2 Ambient

After using Portal 2's in-game level editor for a while I really started to like the background track. In this web app, I wanted to bring these sounds to the web.

I used the original blips and background sounds from Portal 2, so this web app will sound mostly identical.

If you have questions or ideas, please contact me on Twitter.

Volume: 80 %

Blip delay: 2-12 sec.

Background volume: 40 %
